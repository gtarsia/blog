
**HTTP Methods** are sometimes called **HTTP Verbs** too.

There's 9 HTTP Methods. We're gonna explain the following ones:
* GET: to **get** data from the server.
* POST: to **create** data in the server.
* PUT: to **update** data in the server.
* DELETE: to **delete** data in the server.

The following ones also exist, but we won't explain them for now:
* HEAD
* CONNECT
* OPTIONS
* TRACE
* PATH
