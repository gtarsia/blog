---
title: JavaScript types
---

There are two big categories of JavaScript data types:

- Primitives
- Objects

Primitives are the following:

- String: any possible string, for example `"Hello World"`
- Booleans: `true` and `false`
- Number: any possible number, for example `3.53`
- `null`
- `undefined`
- BigInt: for example, `BigInt(748392732394934)`
- Symbol

Objects on the other hand are "containers" that have key/value properties. For example:

```js
{
  name: 'Mike',
  age: 20,
}
```
