---
title: A modern HTML page
---

This is a good, modern HTML [[boilerplate]] that you can build out your sites off.

```html
<!doctype html>
<html lang="en-US">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width" />
  </head>
  <body>
    Hello World!
  </body>
</html>
```

Copy this and put it in a file called `page.html` and open it in chrome (for example, by dragging it to chrome). You should see the text "Hello world" in the page.

Always have your HTML pages have the structure as shown here.
