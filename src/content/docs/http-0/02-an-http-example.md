---
title: An HTTP Example
---

Just to clarify, developer don't manually write these HTTP messages. Instead, what usually happens in professional practice is, clients generate.

A very good example of this is a browser. When you send `example.com` in your address bar, your browser somehow uses a DNS service to translate `example.com` into an IP, and then starts a connection to that IP to which it sends a GET Request. And this is done behind the scenes by your browser.

Another example is using the famous CLI application `curl`. A similar thing would happen to the previous if you run `curl example.com`.
