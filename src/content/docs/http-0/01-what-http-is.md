title: What HTTP is
---

HTTP is the foundation of data communication for the Web.

It is the protocol used to send and receive data in the Web.

HTTP has had three important versions used so far:

1. HTTP/1.1
2. HTTP/2
3. HTTP/3

We'll be discussing HTTP/1.1 here because it is the most widely used historically, and still is as of 2024.

It is based on the TCP/IP protocol (which won't be covered in any of my courses because of it being too lower level).

There are two actors in HTTP:

1. Client.
2. Server.

Communications in HTTP follow the form of:

1. A client sends a **Request Message** to a server.
2. The server sends a **Response Message** back to the client.
