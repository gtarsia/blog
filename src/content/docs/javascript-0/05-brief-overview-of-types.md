---
title: Brief overview of types
---

Let's check a few of JavaScript's common types:

## String

```js
const action = "run";
```

`'run'` is a `String`. It is what we use to represent text in JavaScript

## Number

```
const age = 20
```

`20` is a `Number`.

## Object

```js
const obj = { action: "run", age: 20 };
```

The variable `obj` is now referencing an object. They are also called object literals.

## Array

```js
const arr = ["run", 20];
```

The variable `arr` is now referencing an array. In our case, the array has `'run'` in the position `0` and `20` in the position `1`.
